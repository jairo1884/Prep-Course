// * Variables
Es la forma que tenemos de tomar un valor y guardarlo. Ejemplo: var saludo = "Hola";

// * Strings
Es un tipo de dato, en este caso es una cadena de texto que se asigna entre comillas.

// * Funciones (argumentos, `return`)
una funcion la usamos para calcular algo ue necesitamos en nuestro codigo.

los argumentos de las funciones la asignamos entre los parentesis diciendle a la funcion lo que queremos ejecutar.

return, es na declaracion de una valor que queremos devolver, es el resultado de nuestra funcion contruida.

// * Declaraciones `if`
es un control de flujo de nuestra funcion para que pueda calcular lo que se le solicita que haga.

// * Valores booleanos (`true`, `false`)
son valores binarios, de dos opciones activar o desactivar, falso o verdadero.
